# Nguyen Van Hieu — Portfolio

A highly advanced personal portfolio website built with modern web technologies. Features cinematic animations, 3D galaxy backgrounds, interactive terminal, and AI chat assistant.

## 🚀 Tech Stack

- **Framework**: Next.js 14 (App Router) + TypeScript
- **Styling**: TailwindCSS + CSS custom properties
- **Animation**: Framer Motion + GSAP
- **3D Graphics**: Three.js + React Three Fiber + Drei
- **Smooth Scroll**: Lenis
- **UI**: Custom components with glassmorphism design
- **Fonts**: Sora + JetBrains Mono
